# Traffic, Climate, and Gas Price Prediction ML Project

This project predicts traffic, climate impact, and gas prices along a route using machine learning.

## Structure

- `data/` : Raw and processed datasets
- `notebooks/` : Exploratory analysis
- `src/` : Python scripts (loading, preprocessing, modeling)
- `tests/` : Unit tests

## Usage

1. Load data using `src/data_loader.py`
2. Preprocess data using `src/preprocessing.py`
3. Train model using `src/model.py`
4. Make predictions using `src/predict.py`
